Neste projeto aprendemos a utilizar objetos, com funções.
Foi-se possível gerar uma webpage, na qual ao clicar no botão "Trocar a cor" ele altera a cor do background da page.
Objeto -> É umma coleção de propriedades, sendo uma propriedade uma associação entre um nome(ou chave) e um valor.
Utilizamos o objeto Math, com algumas funções deste, além de usar o novo comando "document.body.style.backgroundColor"
