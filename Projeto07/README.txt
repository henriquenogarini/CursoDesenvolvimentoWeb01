Neste projeto aprendemos a calcular com operadores matemáticos básicos como ( + , -, *, / e %).
Além disso criamos uma calculadora conversora de graus de celsius para fahrenheit.
Aprendemos a utilizar o método parseFloat para o arquivo receber um float e o método toFixec(n)
para que possamos controlar a quantidade de números depois da vírgula de um float.
