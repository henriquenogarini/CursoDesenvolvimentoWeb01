Neste projeto aprendemos a utilizar as tabelas em HTML:
Para criar uma tabela usamos: <table></table>
Para criar a head da tabela usamos: <thead></thead>
Para adicionar itens dentro da tabela row, que é o header da tabela, utilizamos: <tr></tr>
Para colocar itens dentro da table row, utilizamos <th></th>
Para criar o table body, após a inserção do head, utilizamos: <tbody></tbody>
Para adicionar a coluna na tabela: <tr></tr>
Para adicionar um item dentro da coluna: <td></td>
